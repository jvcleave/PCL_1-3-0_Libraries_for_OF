\section{Stretch}

The stretch is the ratio of the minimum edge length to the maximum diagonal, normalized
so that a unit cube has a value of 1:
\[
  q = \sqrt{3}\frac{L_{\min}}{D_{\max}}.
\]

Note that if $D_{\max} < DBL\_MIN$, we set $q = DBL\_MAX$.
