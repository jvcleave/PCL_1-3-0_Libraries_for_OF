\section{Stretch}

The stretch is
\[
q = \frac{ \sqrt{2} \min_{i\in\{0,1,2,3\}}\left\{L_i\right\} }{ D_{\max} }
\]

Note that if $D_{\max} < DBL\_MIN$, we take $q = DBL\_MAX$.
